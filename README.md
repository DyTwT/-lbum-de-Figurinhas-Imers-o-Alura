# 📚 Álbum de Figurinhas - Feminismo

Este projeto foi desenvolvido durante a **Imersão Arquitetura Web com IA**, com o objetivo de criar um álbum de figurinhas digital interativo.

## 💡 Sobre o projeto

O desafio da imersão era desenvolver um álbum com tema livre.  
Escolhi abordar o **feminismo**, inspirado pelas questões atuais relacionadas às mulheres e à importância de dar visibilidade a essas pautas.

## 🚀 Funcionalidades

- 📖 Navegação entre páginas:
  - Clicando nas setas laterais
  - Ou arrastando o mouse

- 🌙 Alternância de tema:
  - Modo claro
  - Modo escuro

- 🔊 Controle de som:
  - Ativar ou desativar o som das páginas

## 🛠️ Tecnologias utilizadas

- HTML
- CSS
- JavaScript

## 🤖 Uso de Inteligência Artificial

Durante o desenvolvimento, utilizei a **IA do Google (Antigravity)** como apoio para construção e aprimoramento do projeto.
